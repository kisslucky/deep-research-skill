# Research Levels

Use three knobs together instead of treating "deep research" as a single mode.

## 1. Time Budget

| Budget | Typical use |
| --- | --- |
| `S` | quick fact check, initial scan, fast triage |
| `M` | shortlist comparison, fast recommendation |
| `L` | option evaluation, structured analysis |
| `XL` | deep decision support, strategy memo, durable report |

## 2. Depth Tier

| Tier | Expectation |
| --- | --- |
| `T1` | confirm what is happening and why it matters |
| `T2` | compare options, surface tradeoffs, recommend a direction |
| `T3` | build a decision memo with evidence, counterpoints, and next steps |

## 3. Deliverable Shape

| Shape | Default sections |
| --- | --- |
| `briefing` | answer, evidence bullets, risks/open questions |
| `standard-report` | executive summary, comparison, recommendation, risks |
| `deep-report` | executive summary, scope, evidence map, implications, recommendation, blind spots, next steps |

## Recommended Combinations

| Mode | Minimum source floor | Best fit |
| --- | --- | --- |
| `S-T1 briefing` | 3-5 sources with at least 1 primary source | fast updates, quick validation |
| `M-T2 standard-report` | 8-12 sources across primary + community + comparison | product, market, and tool selection |
| `L-T2 standard-report` | 10-15 sources with stronger first-party evidence | serious option evaluation |
| `XL-T3 deep-report` | 15+ sources across primary, expert, comparative, and community signals | strategy and decision support |

## Consulting-Style Deep Report Pattern

For `T3` work, structure the report like this:

1. Executive summary
2. Question or decision being supported
3. Key findings by theme
4. Evidence table or source map
5. Counter-evidence and uncertainty
6. Implications for the user
7. Recommendation
8. Blind spots and next steps

## Quality Bar

Treat the run as incomplete if any of these are missing:

- claim-to-source traceability
- explicit separation of fact vs inference
- at least one material tradeoff
- at least one blind spot or missing-data note
- a clear next action
